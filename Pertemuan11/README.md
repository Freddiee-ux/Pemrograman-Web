LaundryCrafty adalah sebuah aplikasi berbasis web yang dirancang untuk membantu pengelolaan usaha laundry agar lebih cepat, efisien, dan terorganisir. Sistem ini dibuat menggunakan PHP Native, MySQL, dan Bootstrap 5 untuk memberikan tampilan yang modern dan mudah digunakan.

Website ini menyediakan berbagai fitur penting yang digunakan oleh admin maupun kasir dalam mengelola aktivitas harian laundry, mulai dari pencatatan pelanggan hingga laporan pendapatan.

1. Manajemen Pelanggan

  Admin dapat menambahkan, mengubah, dan menghapus data pelanggan.
  Data yang disimpan meliputi: nama, alamat, dan nomor telepon.

2. Manajemen Layanan

  Website menyediakan fitur untuk mengelola jenis layanan seperti:
  - Cuci Kering
  - Cuci Setrika
  - Express
  
  Setiap layanan memiliki harga per kilogram yang dapat diatur oleh admin.

3. Transaksi Laundry

  Kasir dapat mencatat transaksi pelanggan, termasuk:
    - memilih pelanggan
    - memilih jenis layanan
    - memasukkan berat cucian
    - menghitung total harga otomatis
    - menentukan tanggal masuk dan tanggal selesai
  
  Terdapat juga fitur untuk memperbarui status cucian:
    - Proses
    - Selesai
    - Sudah Diambil

4. Dashboard Statistik
  
  Halaman dashboard menampilkan:
    - Total pelanggan
    - Total transaksi hari ini
    - Total pendapatan hari ini
    - Data diambil langsung dari database agar selalu real-time.

5. Laporan Pendapatan

  Admin dapat melihat laporan keuangan berdasarkan:
    - filter tanggal
    - pendapatan harian
    - total transaksi pada periode tertentu
    - Laporan ini memudahkan pemilik usaha mengevaluasi performa harian dan bulanan.

6. Keamanan Login

  Sistem dilengkapi autentikasi admin:
    - password di-hash dengan password_hash()
    - session digunakan untuk keamanan halaman
    - tombol logout tersedia

- Teknologi yang Digunakan
  - Komponen	Teknologi
  - Front-End	HTML, CSS, Bootstrap 5
  - Back-End	PHP Native
  - Database	MySQL (phpMyAdmin)
  - Server	Apache (XAMPP)
  - UI Components	Bootstrap Icons
  - Autentikasi	PHP Sessions + Password Hash

- Manfaat Website LaundryCrafty
  - Mempercepat pencatatan transaksi
  - Mengurangi kesalahan perhitungan harga
  - Memudahkan pencarian dan pengelolaan data pelanggan
  - Menyediakan laporan pendapatan yang akurat
  - Menghadirkan antarmuka yang mudah digunakan oleh kasir atau admin
